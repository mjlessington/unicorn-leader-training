## Assessment 6: Psychological Safety Pulse

*Source: Module 3 — Based on Edmondson's research*

Have each team member rate 1-5 (strongly disagree to strongly agree):

| # | Statement | Rating (1-5) |
|---|-----------|:------------:|
| 1 | If I make a mistake on this team, it is **not** held against me | |
| 2 | Members of this team are able to bring up problems and tough issues | |
| 3 | People on this team accept others for being different | |
| 4 | It is safe to take a risk on this team | |
| 5 | It is easy to ask other members of this team for help | |
| 6 | No one on this team would deliberately undermine my efforts | |
| 7 | Working with this team, my unique skills and talents are valued | |

**Average Score: ___/5.0**

| Score | Interpretation |
|:-----:|---------------|
| 4.0+ | Strong safety — sustain and model |
| 3.0-3.9 | Moderate safety — targeted improvement needed |
| Below 3.0 | Low safety — urgent leadership attention required |
